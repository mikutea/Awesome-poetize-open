# 联系方式管理功能说明

## 功能概述

通过**资源聚合**功能管理联系方式，添加后的联系方式将在首页的"朋友圈"按钮下方以图标形式展示。支持添加微信、QQ、B站、力扣、QQ邮箱、GitHub等各种联系方式。

## 使用方法

### 1. 进入资源聚合管理

1. 以站长身份登录后台管理系统
2. 在侧边栏点击"资源聚合"菜单
3. 点击"新增资源聚合"按钮

### 2. 添加联系方式

在弹出的对话框中：

#### 第一步：选择类型
- 点击顶部的**"联系方式"**单选按钮

#### 第二步：填写信息

- **标题**（必填）：联系方式的名称
  - 例如：微信、QQ、B站、力扣、GitHub、邮箱等

- **分类**（可选）：联系方式的类型分类
  - 例如：社交媒体、邮箱、技术平台等
  - 用于后续筛选和管理

- **简介**（可选）：联系方式的描述
  - 例如：扫码添加微信、关注我的B站账号等
  - 鼠标悬停时会显示

- **封面**（推荐）：联系方式的图标
  - 方式1：直接输入图片URL
  - 方式2：点击"上传封面"按钮上传图片
  - 方式3：**直接粘贴SVG代码**（系统会自动上传并返回URL）✨
  - 建议使用正方形图标，大小不超过2MB
  - 推荐尺寸：25x25 像素

- **链接**（可选）：点击图标后跳转的地址
  - 例如：
    - 微信：可以上传微信二维码图片链接
    - QQ：`https://qm.qq.com/cgi-bin/qm/qr?k=你的QQ分享码`
    - B站：`https://space.bilibili.com/你的UID`
    - GitHub：`https://github.com/你的用户名`
    - 邮箱：`mailto:your-email@example.com`
  - 如果不填写，点击图标将无响应

- **资源类型**：自动显示为"contact"

- **备注**（可选）：内部备注信息

#### 第三步：提交保存
- 点击底部的"提交"按钮保存

### 3. 管理联系方式

#### 查看联系方式列表
- 在资源聚合页面的筛选器中选择"联系方式"类型
- 点击"搜索"按钮查看所有联系方式

#### 编辑联系方式
1. 在列表中找到要编辑的联系方式
2. 点击"编辑"按钮
3. 修改相关信息
4. 点击"提交"保存

#### 启用/禁用联系方式
- 在列表中找到对应的联系方式
- 点击状态旁边的开关按钮
- 只有启用状态的联系方式会在首页显示

#### 删除联系方式
1. 在列表中找到要删除的联系方式
2. 点击"删除"按钮
3. 确认删除操作

### 4. 前台展示

联系方式会自动显示在首页侧边栏的位置：
- **位置**：在"朋友圈"按钮的下方
- **样式**：圆形图标形式展示
- **动效**：鼠标悬停时图标会向上浮动
- **交互**：
  - 点击有链接的图标会在新窗口打开对应链接
  - 鼠标悬停会显示标题和简介
  - 没有链接的图标点击无响应

## 技术实现

### 数据存储
- 使用现有的 `resource_path` 数据库表
- `type` 字段值为 `"contact"` 表示联系方式类型
- 无需额外建表，充分利用现有资源聚合功能

### 前端组件
- **管理界面**：`poetize-ui/src/components/admin/resourcePathList.vue`
- **展示界面**：`poetize-ui/src/components/myAside.vue`

### API接口
使用现有的资源聚合接口：
```
POST /webInfo/listResourcePath
参数：{
  current: 1,
  size: 100,
  resourceType: "contact",
  status: true
}
```

## 注意事项

1. **权限要求**：只有站长（userType=0）才能管理资源聚合功能
2. **图标建议**：
   - 使用正方形图片
   - 推荐尺寸：25x25 像素
   - 文件大小不超过 2MB
   - 格式支持：jpg、png、gif、webp、**svg** 等
3. **SVG使用**：
   - ✨ 支持直接粘贴SVG代码到封面字段
   - 系统会自动检测并上传为SVG文件
   - 自动返回文件URL，无需手动操作
4. **排序**：联系方式按照创建时间显示，后续可以通过数据库调整顺序
5. **状态控制**：只有启用状态的联系方式才会在首页展示
6. **链接安全**：建议使用 HTTPS 链接

## 优势

✅ **无需额外开发**：利用现有资源聚合功能

## 常见问题

### Q1：联系方式没有显示在首页？
**A**：检查以下几点：
- 确认联系方式的状态为"启用"
- 确认已上传封面图片或设置了图标URL
- 刷新页面或清除浏览器缓存

### Q2：如何调整联系方式的显示顺序？
**A**：目前按创建时间排序，如需调整可以：
- 删除后重新添加（按需要的顺序）
- 或直接在数据库中调整 `create_time` 字段

### Q3：可以添加多少个联系方式？
**A**：理论上没有数量限制，但建议不超过10个，以保持页面简洁美观。

### Q4：图标显示不清晰？
**A**：建议使用更高分辨率的图标，推荐 100x100 像素或更大。

### Q5：点击图标没有反应？
**A**：检查是否填写了"链接"字段，如果没有填写链接则点击无效。
